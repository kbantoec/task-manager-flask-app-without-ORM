# Task Manager App (without ORM)

Task Manager App without the `SQLAlchemy` Module.

* `tmapp` stands for **t**ask **m**anager **app**

Workflow:

```powershell
git init
mkdir tmapp
pip list
pip install flask
touch README.md, .gitignore
cd .\tmapp\
mkdir templates, static, tests
touch __init__.py, views.py, models.py
cd ..
```
